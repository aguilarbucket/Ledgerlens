# LedgerLens brand assets

These are the curated production assets selected from the owner-provided LedgerLens Brand Pack v2
on 2026-07-19. The original ZIP and extracted source pack remain local and are intentionally
excluded from Git and Docker build context.

| Asset | Product use |
| --- | --- |
| `ledgerlens-app-icon-192.png` | Streamlit page icon and browser identity |
| `ledgerlens-header-lockup.png` | Dark application header and dark-mode README |
| `ledgerlens-lockup-navy.png` | Light-mode README and light surfaces |
| `ledgerlens-avatar-512.png` | GitHub/Docker Hub profile candidate |
| `ledgerlens-brand-board.png` | Visual reference for mark, palette, and variants |

Core colors are `#08111F`, `#4F8CFF`, `#2DD4BF`, `#9B87F5`, and `#F8FAFC`. Performance green is
not a brand color and remains reserved for positive financial values. Preserve the supplied clear
space and proportions; do not stretch, rotate, recolor, or place the mark on noisy imagery.
