# Demo script

Target duration: under three minutes. This script describes the current slice and will be expanded
when invoice extraction and the analyst reports are implemented.

1. State the problem: manual brokerage invoices do not create a trustworthy, explainable ledger.
2. Open LedgerLens and point out the synthetic-demo notice.
3. Show portfolio value, unrealized P/L, and price coverage.
4. Show that repeated purchases are consolidated using weighted average cost.
5. Open Purchase history and show synthetic document references.
6. Explain that all calculations are deterministic Python outputs and that AI narrative is kept
   separate from financial truth.

Do not record a final submission video until the complete P0 flow is implemented and verified.
